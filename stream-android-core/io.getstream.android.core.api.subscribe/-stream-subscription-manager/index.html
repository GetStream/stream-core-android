StreamSubscriptionManager

interface StreamSubscriptionManager<T> : StreamObservable<T>

Registry that owns a set of listeners and provides minimal lifecycle control over them.

Typical usage:

private val subs = StreamSubscriptionManagerImpl<(Event) -> Unit>()

fun onSomethingHappened(event: Event) {
    subs.forEach { listener -> listener(event) }
}

Thread-safety: Implementations must be thread-safe. subscribe, clear, and forEach can be invoked from different coroutines or threads. Implementations must not hold locks while invoking user code passed to forEach, to avoid deadlocks or long pauses.
