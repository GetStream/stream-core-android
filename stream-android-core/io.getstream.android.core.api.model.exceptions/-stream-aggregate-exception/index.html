StreamAggregateException

class StreamAggregateException(message: String = "", val causes: List<Throwable>) : StreamClientException

Exception representing multiple errors that occurred during a single operation.

This is useful when a single logical operation results in multiple underlying failures, such as parallel network requests or batch processing. It aggregates the individual exceptions for later inspection.
