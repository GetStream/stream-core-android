StreamEventAggregationPolicy

data class StreamEventAggregationPolicy<T>

Complete aggregation configuration for StreamEventAggregator.

Combines behavior (how to extract types and deserialize) with tuning (thresholds, windows, queue capacity). Once created via a factory method, the policy is guaranteed valid — all invariants are checked at construction time.

The primary constructor is private. Create instances through the from factory method.

Invariants

  • aggregationThreshold > 0

  • maxWindowMs > 0

  • dispatchQueueCapacity > 0

Parameters

T

The deserialized event type.

extractType

Extracts the event type string from a raw message (typically JSON). Returns null if the type cannot be determined — events with unknown type are grouped under an empty key.

deserialize

Deserializes a raw message into T. Returns Result.failure on parse errors.

aggregationThreshold

Number of accumulated events that triggers aggregated delivery instead of individual dispatch.

maxWindowMs

Maximum time (milliseconds) the collector will wait before packaging and delivering whatever has accumulated. This is the latency ceiling.

dispatchQueueCapacity

Bounded capacity of the dispatch queue between collector and dispatcher. When full, the collector logs a warning.
