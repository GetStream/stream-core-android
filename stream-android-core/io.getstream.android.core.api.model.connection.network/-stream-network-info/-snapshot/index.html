Immutable capture of the device's currently selected network at a specific point in time.

Every field maps back to values provided by android.net.NetworkCapabilities or android.net.LinkProperties. Nullable fields either were not reported by the platform or are hidden due to missing runtime permissions (for example, fine location for Wi-Fi SSID).
