StreamJsonSerialization

interface StreamJsonSerialization

General contract for JSON serialization and deserialization.

This interface abstracts away the actual JSON parsing engine (e.g. Moshi, Gson, kotlinx.serialization), allowing the SDK to plug in a specific implementation without exposing the underlying library.

Implementations are expected to provide robust error handling and return results wrapped in Result.
