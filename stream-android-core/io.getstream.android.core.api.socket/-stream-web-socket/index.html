interface StreamWebSocket<T : StreamWebSocketListener> : StreamSubscriptionManager<T>

Represents a WebSocket connection used for real-time communication with the Stream API.

Implementations of this interface manage the lifecycle of a socket connection, handle sending messages, and provide subscription management for socket events.

Parameters

T

The type of listener used to receive WebSocket events.
