StreamTelemetryConfig

data class StreamTelemetryConfig(val root: File? = null, val basePath: String = "stream/telemetry", val version: String, val memoryCapacity: Int = 500, val diskCapacity: Long, val redactor: StreamSignalRedactor? = null)

Configuration for StreamTelemetry.

Parameters

root

Root directory for disk spill storage. Defaults to context.cacheDir (passed at creation time via the factory).

basePath

Subdirectory under root for telemetry files (e.g. "stream/telemetry").

version

Version tag for the disk format — typically the core SDK version. On initialization, any directories under {root}/{basePath} that don't match this version are deleted.

memoryCapacity

Maximum number of signals held in memory per scope before spilling to disk.

diskCapacity

Maximum bytes of disk storage per scope. When exceeded, the oldest signals are dropped.

redactor

Optional StreamSignalRedactor applied to every signal on emit.
